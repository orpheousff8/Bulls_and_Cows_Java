Managing a donut store inventory

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Create a Java program that contains two classes: one class named 'Donut' and another class named 'DonutStore'. The 'Donut' class has two private data variables: 'flavor' and 'price'. The 'DonutStore' class has a public method named 'sellDonut' that takes as input a flavor (as a string) and returns the price of the donut (as a double). For our purposes, a 'chocolate' donut is priced at $1.50 and a 'vanilla' donut is priced at $1.00. Other flavors are not available, if an unavailable flavor is requested, return 0.0. The 'DonutStore' should interact with 'Donut' objects utilizing an appropriate level of access control. Once the program is running, the user should enter a flavor and the program should return the corresponding price, or 0.0 if the flavor does not exist.
Sample Input:
chocolate

Sample Output:
1.5


Sample Input:
vanilla

Sample Output:
1.0



Memory limit: 256 MB
Time limit: 5 seconds
