Printing a given message with class names

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
You're tasked with creating two classes: 'Package1.ClassAlpha' and 'Package2.ClassBeta'. Each class should have a method 'display' that prints the class name followed by the message gotten from the standard input. Your task is to accept a single line of input. The input is a string that will be used by the 'display' methods. If the input string is 'Hello', then 'ClassAlpha' should print 'ClassAlpha: Hello' and 'ClassBeta' should print 'ClassBeta: Hello'.
Sample Input:
Hello

Sample Output:
ClassAlpha: Hello
ClassBeta: Hello


Sample Input:
Java

Sample Output:
ClassAlpha: Java
ClassBeta: Java



Memory limit: 256 MB
Time limit: 5 seconds
