Creating and testing inherited vehicle and car classes

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
You will design a vehicle class that gets number of tires and color as input, and create a car class that inherits from vehicle, with an extra field: speed. Both classes have a toString method to print their features. If we create an instance of car class, it should print its tire number, color and speed. Create your classes and test with some instances of car class.
Sample Input:
4
Blue
120

Sample Output:
Tires: 4, Color: Blue, Speed: 120


Sample Input:
2
Red
80

Sample Output:
Tires: 2, Color: Red, Speed: 80



Memory limit: 256 MB
Time limit: 5 seconds
